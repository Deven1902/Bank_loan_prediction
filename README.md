## Usage:-

- Clone the repo.
- Open terminal in working directory.
- Run following commands.

  ```
  pip install -r requirements.txt
  ```

- `app.py` is the main Streamlit file of our app. 
- To run app, write following command in terminal.
  
  ```
  streamlit run app.py
  ```
